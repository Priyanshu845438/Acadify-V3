### **4. Mobile Responsive Instructions**

The Bootstrap grid handles the core responsiveness, but we should refine it for the best mobile experience.

*   **Layout Stacking:** On screens smaller than the `lg` breakpoint (`<992px`), the two `col-lg-6` columns will automatically stack. The **Image Column will appear below the Text Content Column**. This is the ideal, logical flow for a mobile device.
*   **Text Alignment:** On mobile, the left-aligned text can sometimes feel unbalanced. To fix this, you can add responsive text-alignment classes to the parent column:
    *   Change `<div class="col-lg-6">` to `<div class="col-lg-6 text-center text-lg-start">`. This will center all the text and buttons on mobile and tablet, then snap it back to left-aligned on desktop.
*   **Button Wrapping:** The `flex-wrap` class on the button container (`d-flex flex-wrap`) is important. If the screen is too narrow for all three buttons to fit side-by-side, they will wrap gracefully onto the next line instead of overflowing.
*   **Image:** The `.img-fluid` class ensures the image scales down perfectly to fit its container on any screen size.
